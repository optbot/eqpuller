eqpuller
===
Maintain a list of active equities for which options
are to be retrieved.

Usage
---
### Basic
1.  Install:
    
        $ npm install
        $ sudo npm run setup

1.  Configure. To override defaults cf. [Configuration](#Configuration) below.

1.  Start:

        $ sudo npm start

1.  Stop:

        $ sudo npm stop
